## Midterm Examination Reflection

### 1. What is Git used for?

Git is used to keep track of changes made to files in a project. It helps save different versions of our work and makes it easier to manage updates.

### 2. What is the difference between Git and GitHub?

Git is a tool that helps us manage our project on your computer. GitHub is a website where we can store our Git projects online and share them with others.

### 3. Why is branching useful?

Branching allows developers to work on new features or make changes without affecting the main project. It also makes it easier to test changes before merging them into the main branch.

### 4. What command did you use to upload your local commits to GitHub?

I used the `git push` command to upload my local commits to GitHub. But if it was my first time uploading, I used `git push -u origin main` to connect the local branch to Github Repository.

### 5. What did you learn from this activity?

I learned how to initialize a Git repository, create commits, use branches, and merge changes. I also learned how to connect a local project to GitHub and upload my work successfully.
